# Boot and Jekyll

```
rvm default 2.3.3
get install jekyll
npm i -g bower
bower install bootstrap#4.0 --force-latest --save
cp scss, js from bower_components/ into project assets/css and assets/js
jekyll build
jekyll serve
```
